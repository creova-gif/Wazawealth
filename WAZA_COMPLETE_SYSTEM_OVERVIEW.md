## 🎨 Design Philosophy (Executed)

### 1. Confidence over Excitement ✅
- 90% black & white color palette
- Large hero numbers (text-7xl, 72px)
- Calm animations (400-600ms)
- No hype, no fear-based copy

### 2. Guidance over Speculation ✅
- AI insights (educational, not predictive)
- "On track" vs "You're winning!"
- Confidence bands (not guarantees)
- Plain language (no jargon)

### 3. Goals over Stock Picking ✅
- Purpose-based accounts first
- "Everyday Growth" not "Account #123"
- Progress bars toward goals
- Dashboard shows wealth, not just stocks

### 4. Simplicity over Complexity ✅
- One question per screen
- Progressive disclosure
- Clear visual hierarchy
- 8px spacing scale (consistent)

## 🎨 Complete Design System

### Color Palette (90/8/2 Rule)

**90% Black & White**:
```
Black:   #000000   Primary text, CTAs
White:   #FFFFFF   Backgrounds, cards
Zinc-50: #FAFAFA   Subtle backgrounds
```

**8% Grays** (Zinc):
```
100: #F4F4F5   Inactive states
200: #E4E4E7   Borders (default)
300: #D4D4D8   Borders (hover)
400: #A1A1AA   Borders (active)
500: #71717A   Secondary text
600: #52525B   Primary gray text
700: #3F3F46   Dark gray
800: #27272A   Very dark gray
900: #18181B   Near black
```

**2% Accent** (Semantic):
```
Green-50:  #ECFDF5   Success backgrounds
Green-600: #16A34A   Success text
Blue-50:   #EFF6FF   AI backgrounds
Blue-700:  #1D4ED8   AI text
Amber-50:  #FFFBEB   Warning backgrounds
Amber-700: #B45309   Warning text
Red-600:   #DC2626   Error (rare)
```

### Typography Scale

```
Hero Wealth:    text-7xl (72px)  font-light (300)
Large Number:   text-5xl (48px)  font-light (300)
Card Number:    text-2xl (24px)  font-light (300)
Section Title:  text-2xl (24px)  font-light (300)
Card Title:     text-lg (18px)   font-medium (500)
Body:           text-base (16px) font-normal (400)
Caption:        text-sm (14px)   font-normal (400)
Micro Label:    text-xs (12px)   uppercase tracking-wide
```

---

### Spacing System (8px Base)

```
2px   0.5    Dividers
4px   1      Icon-text gaps
8px   2      Tight spacing
12px  3      Card inner padding
16px  4      Standard gaps
24px  6      Card padding, sections
32px  8      Large card padding
48px  12     Section breaks
64px  16     Screen-level spacing
```

---

### Motion System

**Timings**:
```
Fast:     100-200ms   Button feedback
Standard: 400ms       Page transitions
Slow:     600ms       Complex animations
```

**Easing**:
```
ease-out:     Default (0, 0, 0.2, 1)
ease-in-out:  Modals (0.4, 0, 0.2, 1)
spring:       Success states
```

**Common Patterns**:
```tsx
// Entry (fade + slide)
initial={{ opacity: 0, y: 20 }}
animate={{ opacity: 1, y: 0 }}
transition={{ duration: 0.4, ease: 'easeOut' }}

// Stagger (lists)
transition={{ delay: index * 0.1 }}

// Button tap
whileTap={{ scale: 0.97 }}

// Expand/collapse
initial={{ height: 0, opacity: 0 }}
animate={{ height: 'auto', opacity: 1 }}
transition={{ duration: 0.3 }}
```

## 🎓 Key Learnings

### What Works
1. **Big numbers matter** — 7xl creates instant clarity
2. **Calm colors win** — 90/8/2 rule feels premium
3. **Bilingual from start** — Easier than retrofitting
4. **Progressive disclosure** — Don't show everything at once
5. **AI as guide** — Not predictor, not salesperson

### What to Avoid
1. **Red/green everywhere** — Use sparingly
2. **Trading jargon** — Plain language always
3. **Too many CTAs** — Max 2-3 per screen
4. **Fear-based copy** — "Don't miss out!" doesn't fit
5. **Complexity** — Simple > clever
